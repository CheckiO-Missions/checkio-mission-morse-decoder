Państwa zadaniem jest odszyfrowanie tajnej wiadomości za pomocą strony Morse code.
Wiadomość będzie składać się ze słów z 3 spacjami między nimi i 1 spacją między każdą literą każdego słowa.
Jeśli odszyfrowany tekst zaczyna się od jakiejś litery, musi ona zostać zapisana wielkimi literami.

Dane wejściowe: Tajna wiadomość (str{% if is_js %}ing{% endif %}).

Wyjście: Odszyfrowany tekst (str{% if is_js %}ing{% endif %}).

Przykłady: 

{{init_code_tmpl}}

Jak jest używany: Do kryptografii i pracy szpiegowskiej.

Warunki wstępne:

  • 0 < len(message) < 100;
  • Wiadomość będzie składać się wyłącznie z cyfr i angielskich liter.
