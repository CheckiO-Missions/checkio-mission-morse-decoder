Twoim zadaniem jest odszyfrowanie wiadomości zaszyfrowanej kodem Morse'a.
Wiadomość składa się z słów oddzielonych trzema spacjami. Litery w słowach oddzielone są pojedynczą spacją.
Wiadomość powinna zaczynać się wielką literą, jeśli pierwszym znakiem nie jest cyfra.

Wejście: String - zaszyfrowana wiadomość.

Wyjście: String - odszyfrowana wiadomość.

Example:

{% if interpreter.slug == "js-node" %} 
morseDecoder("... --- -- .   - . -..- -") == "Some text"
morseDecoder("..--- ----- .---- ---..") == "2018"
morseDecoder(".. -   .-- .- ...   .-   --. --- --- -..   -.. .- -.--") == "It was a good day"
{% else %} 
morse_decoder("... --- -- .   - . -..- -") == "Some text"
morse_decoder("..--- ----- .---- ---..") == "2018"
morse_decoder(".. -   .-- .- ...   .-   --. --- --- -..   -.. .- -.--") == "It was a good day"
{% endif %}

Do czego jest używane: W kryptografii i szpiegostwie.

Warunek wstępny::
0 < len(message) < 100
Wiadomość składa się wyłącznie z liter alfabetu angielskiego i liczb.
